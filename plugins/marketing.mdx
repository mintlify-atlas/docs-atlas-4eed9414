```bash
claude plugins add knowledge-work-plugins/marketing
```

### /draft-content

Generate marketing content drafts tailored to your brand and audience.

**Usage:**
```
/draft-content <content type and topic>
```

**Inputs you'll provide:**
- **Content type:** Blog post, social media, email newsletter, landing page, press release, or case study
- **Topic:** The subject or theme
- **Target audience:** Who this is for (role, industry, pain points)
- **Key messages:** 2-4 main takeaways
- **Tone:** Authoritative, conversational, technical, etc.
- **Length:** Target word count or format constraint

**Content types supported:**

**Blog Post**
- Engaging headline (2-3 options)
- Introduction with hook
- 3-5 organized sections with SEO-optimized subheadings
- Conclusion with clear CTA
- Primary keyword suggestions

**Social Media Post**
- Platform-appropriate format and length
- Hook in first line
- Hashtag suggestions (3-5 relevant)
- Engagement prompt
- Platform-specific best practices (LinkedIn, Twitter/X, Instagram, Facebook)

**Email Newsletter**
- Subject line options (2-3 with open-rate considerations)
- Preview text
- Body sections with clear hierarchy
- CTA button text

**Landing Page Copy**
- Headline and subheadline
- Hero section copy
- 3-4 benefit-driven value propositions
- Primary and secondary CTAs
- FAQ section suggestions
- Meta title and description

**Press Release**
- Headline following PR conventions
- Dateline and location
- Lead paragraph (who, what, when, where, why)
- Quote placeholders
- Standard formatting

**Case Study**
- Title emphasizing results
- Customer overview (industry, size, challenge)
- Challenge, solution, and results sections
- Metrics and customer quote placeholders

### /campaign-plan

Generate comprehensive campaign briefs.

**Usage:**
```
/campaign-plan
```

**Example:**
```
> /campaign-plan
Goal: Drive 500 signups for our new product launch
Audience: Technical decision-makers at enterprise companies
Timeline: 6 weeks
Budget range: $20,000-$30,000
```

**What you get:**
- Campaign objectives and audience segmentation
- Key messages and positioning
- Channel strategy
- Week-by-week content calendar
- KPIs and success metrics
- Budget allocation recommendations

### /brand-review

Review content against your brand guidelines.

**Usage:**
```
/brand-review
[paste your draft content]
```

If your brand style guide is configured in local settings, Claude will check your content against:
- Brand voice attributes
- Tone consistency
- Terminology and style guidelines
- Messaging pillars

If not configured, Claude will provide a generic review for clarity, consistency, and professionalism.

## Configuration

Configure your brand voice, style guide, and target personas in a local settings file for personalized output. This allows commands like `/draft-content` and `/brand-review` to automatically apply your brand standards without prompting each time.

**Example settings file location:** `marketing/.claude/settings.local.json`

```json
{
  "brand": {
    "voice": "Authoritative but approachable",
    "tone_attributes": ["Professional", "Helpful", "Clear"],
    "terminology": {
      "preferred": ["AI-powered", "customer success"],
      "avoid": ["bleeding-edge", "game-changing"]
    }
  },
  "target_personas": [
    {
      "name": "Marketing Director",
      "industry": "B2B SaaS",
      "pain_points": ["Attribution", "Proving ROI"]
    }
  ]
}
```

## Example Workflows

### Drafting a Blog Post

```
> /draft-content
Type: blog post
Topic: How AI is transforming B2B marketing
Audience: Marketing directors at mid-market SaaS companies
Key messages: AI saves time on repetitive tasks, improves personalization, requires human oversight
Tone: Authoritative but approachable
Length: 1200 words
```

Claude generates a structured blog post with engaging headline, SEO-optimized subheadings, and clear CTA.

### Planning a Campaign

```
> /campaign-plan
Goal: Drive 500 signups for our new product launch
Audience: Technical decision-makers at enterprise companies
Timeline: 6 weeks
Budget range: $20,000-$30,000
```

Claude produces a campaign brief with objectives, audience segmentation, channel strategy, week-by-week content calendar, and KPIs.

### Reviewing Brand Consistency

```
> /brand-review
[paste your draft content]
```

Claude checks your content against configured brand voice, tone, terminology, and messaging pillars.
